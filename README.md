## 1.第一章为request的应用
request 模块
原生的网络请求的模块，功能非常强大，简单便捷，效率极高
作用：模拟浏览器请求
如何使用：（requests模块的编码流程）
    -指定url
    -发起请求
    -获取相应数据
    -持久化存储
